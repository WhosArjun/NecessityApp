Necessity is a Desktop Application hopefully soon to be IOS App Store App
https://necessity-app.vercel.app/
The app is comprised of two functions Weather System and GPS Navigational System

The Weather System uses Open-Meteo data, and has function such as UV Index, time of sunset and sunrise, and your also able to add many locations that you would like to view the weather of.

The GPS System allows you to either enter a starting location or leave your location as the starting location. You are able to view local places, ETA, and different routes you can take to optimize your travel.
